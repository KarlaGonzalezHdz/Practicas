Hojas de estilo CSS

Cascade Style Sheet, Hojas de estilo en cascada.

Las hojas de estilo es una tecnologia que nos permite controlar la apariencia de una página web.

El código HTML no pone atención en la apariencia del documento , si no en la estructura.

Con CSS podemos especificar estilos como el tamaño de fuente, espaciado entre textos y recuadros.
